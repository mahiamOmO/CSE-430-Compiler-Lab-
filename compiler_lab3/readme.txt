Lab 3: Lexical Analyzer using Flex
===================================

Student: Mahia Akter Momo
Course: CSE 430 - Compiler Design
University: University of Asia Pacific

Files Submitted:
-----------------
1. lexer.l   - Lex source file (lexical analyzer rules)
2. lex.yy.c  - C file generated by flex from lexer.l
3. test.txt  - Sample C input file used for testing

How to Run:
-----------
1. flex lexer.l
2. gcc lex.yy.c -o lexer -lfl
3. ./lexer < test.txt

Description:
------------
This lexical analyzer identifies the following tokens:
- Keywords    : 32 C keywords
- Identifier  : variable/function names
- Constant    : numeric values
- Parenthesis : ( ) { } [ ]
- Punctuation : ; : ,
- Arithmetic  : + - * /
- Logical Op  : > >= < <= == !=
- Assign Op   : =
- Comment     : single line (//)
